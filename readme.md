## Deskripsi Singkat
**RKALearning** adalah sebuah landing page edukasi modern yang dirancang untuk memberikan pengalaman belajar yang interaktif dan responsif bagi siswa tingkat SD, SMP, dan SMA.

---

## Fitur Utama

### 1. Arsitektur Komponen Dinamis
Menggunakan Vanilla JavaScript untuk mengelola dan merender paket belajar secara dinamis berdasarkan jenjang pendidikan (SD/SMP/SMA) tanpa memerlukan *refresh* halaman.

### 2. Desain Responsif & Premium
Implementasi tata letak (*layout*) menggunakan sistem modern **CSS Grid** dan **Flexbox**, memastikan tampilan optimal pada berbagai perangkat mulai dari desktop hingga ponsel pintar.

### 3. Sistem Interaksi Pengguna
- **Formulir Kontak**: Integrasi validasi sisi klien sederhana dengan umpan balik visual melalui modal kustom.
- **Navigasi Sticky**: Mempermudah akses menu utama saat pengguna melakukan *scrolling*.
- **Micro-animations**: Efek hover dan transisi halus untuk meningkatkan *user engagement*.

### 4. Optimalisasi Aset
Penggunaan *placeholder* gambar dinamis dari Lorem Picsum untuk demonstrasi konten yang cepat dan variatif.

## Struktur Proyek

```bash
.
├── index.html   # Struktur utama dan logika aplikasi
└── style.css    # Definisi gaya visual dan sistem desain
```

- **index.html**: Berisi struktur semantik dan blok JavaScript internal untuk manajemen state paket belajar.
- **style.css**: Mengimplementasikan sistem desain berbasis variabel CSS (`:root`) untuk kemudahan kustomisasi tema.
